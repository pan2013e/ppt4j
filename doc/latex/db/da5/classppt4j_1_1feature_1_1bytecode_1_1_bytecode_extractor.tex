Parses the methods of the class by analyzing their bytecode instructions. Skips native or abstract methods, methods without line number info, and $<$clinit$>$ methods. Performs \doxylink{classppt4j_1_1analysis_1_1bytecode_1_1_arg_type_analysis}{Arg\+Type\+Analysis}, \doxylink{classppt4j_1_1analysis_1_1bytecode_1_1_loop_analysis}{Loop\+Analysis}, and clustering of instructions for each method. Generates features for each instruction and removes RETURN instructions if present in $<$clinit$>$ method. Sets is\+Parsed flag to true after parsing is complete.
